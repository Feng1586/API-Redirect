		OpenAI 多模态响应接口：该接口使用OpenAI Responses API 格式提供给用户，该接口的上游端点同样兼容OpenAI Responses API 格式，接口为/v1/responses，当用户调用我们提供的转发端口后，需要获取用户使用的是什么模型，以及消耗的输入输出tokens的数量，并记录其用量以及为用户计费。并且当上游接口返回错误时，也需要返回给用户
